#### 数据库登录访问授权
1. 登录mysql  
    mysql -u root -p密码
2. 使用数据库
    use mysql;
3. 新创建一个用户
    create user 'root'@'%' identified by '123456';
4. 给用户最大授权
    grant all privileges on *.* to 'root'@'%' identified by '123456';
5. 刷新权限表
    LUSH PRIVILEGES;
